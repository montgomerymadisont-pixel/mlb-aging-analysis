
# MLB Player Aging Analysis

## Overview
This project analyzes how age impacts player performance across different baseball positions using historical MLB data.

## Dataset
- 38,000+ player records
- Historical MLB statistics dataset

## Methods
- Feature engineering (custom performance score)
- Regression modeling
- Train/test split and cross-validation

## Results
- R² ≈ 0.69
- Peak performance typically occurs in mid-to-late 20s
- Catchers show earlier performance decline than other positions

## Key Insight
Age and position are strong predictors of performance, but additional factors (injuries, training, team context) also play a role.
